# 青岛市教育局新搞的饭卡的交易查询

## 简介 
青岛市教育局新搞的饭卡的交易查询
    
知道学籍号即可查询

## 使用方法
自行安装PHP，并打开CURL扩展

然后
```bash
php main.php 学籍号
```

比如
```bash
php main.php 2017370201880930***
```

查询学籍号为2017370201880930**的交易记录
