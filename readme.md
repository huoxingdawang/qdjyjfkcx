#青岛市教育局新搞得饭卡的交易查询

##简介 
	青岛市教育局新搞得饭卡的交易查询
    
	知道学籍号即可查询

	目前无法展示详细交易记录，因为我也不知道长啥样。。。。
##使用方法
自行安装PHP，并打开CURL扩展

然后
```bash
php main.php 学籍号 查询时间范围
```

比如
```bash
php main.php 2017370201880930*** 201908
```

查询学籍号为2017370201880930***的同学在2019年08月的交易记录
